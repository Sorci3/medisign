 Summary of the plan:

  Notebook 1 — notebooks/01_preprocessing.ipynb (Tasks 2–4)
  - Explores class distribution and frame lengths
  - Removes 4 corrupt instances (T=0)
  - Converts skeleton keypoints → 3-channel Gaussian heatmaps (3, T=32, H=64, W=64) using body + left/right hands
  (face's 478 landmarks are excluded to keep input compact)
  - 4× augmentation per training sample: original + horizontal flip + temporal flip + jitter
  - Saves X_train.npy, y_train.npy, X_test.npy, y_test.npy to src/dataset/preprocessed/

  Notebook 2 — notebooks/02_i3d_training.ipynb (Tasks 5–8)
  - Custom PoseHeatmapDataset with WeightedRandomSampler to address severe class imbalance
  - Full Inception-I3D architecture (Unit3D → InceptionModule → InceptionI3D)
  - Training: Adam + ReduceLROnPlateau + early stopping + class-weighted cross-entropy
  - Evaluation: accuracy, macro-F1, per-class classification report, confusion matrix
  - save_model() / load_model() functions + inference example
